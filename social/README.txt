visakha.mp3
เก่งมาก นี่คือวันวิสาขบูชา วันที่พระพุทธเจ้าประสูติ ตรัสรู้ และปรินิพพาน

asalha.mp3

christmas.mp3

ramadan.mp3
เก่งมาก นี่คือเดือนรอมฎอน เดือนแห่งการถือศีลอดของชาวมุสลิม


วิธีใช้
1. ใส่ภาพ JPG ให้ครบในโฟลเดอร์ images
2. สร้างเสียง MP3 และใส่ในโฟลเดอร์ audio/clue และ audio/answer
3. เปิด index.html ด้วย Chrome หรือ Edge
4. ถ้าจะนำขึ้น GitHub Pages ให้อัปโหลดทั้งโฟลเดอร์ โดยรักษาชื่อไฟล์และโครงสร้างเดิม
